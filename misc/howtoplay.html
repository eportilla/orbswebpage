HOW TO PLAY

INPUT ACTION
GENERAL
LEFT MOUSE PRESS MOVE ORB
RIGHT MOUSE PRESS ROTATE CAMERA
MIDDLE MOUSE CLICK MOVE ORB TO TARGET POSITION
MOUSE WHEEL ZOOM / CHANGE PREPARED MODULE LEVEL
SHIFT KEY TOGGLE USER INTERFACE
ANY KEY DOWN PREPARE MODULE FROM HOT KEY
ANY KEY UP ACTIVATE PREPARED MODULE
ANY KEY LOAD AN EFFECTIVE LOOTED MODULE AND ASSIGN HOT KEY
TAB KEY LOAD THE LOOTED MODULE
F12 KEY TAKE SCREENSHOT
ENTER KEY ACTIVATE LOCAL CHAT, SEND CHAT MESSAGE
ESCAPE KEY QUIT THE GAME
ON LOOTED MODULE
LEFT MOUSE CLICK LOAD THE LOOTED MODULE
RIGHT MOUSE CLICK GIVE MODULE TO NEAREST ORB
LONG LEFT MOUSE KEY PRESS PUT MODULE IN HOME INVENTORY
LONG RIGHT MOUSE KEY PRESS DESTROY MODULE
CONTROL + RIGHT MOUSE CLICK ADD OR REMOVE MODULE TYPE TO IGNORE LIST
ON LOADED MODULE
ANY KEY CHANGE MODULE HOT KEY
LEFT MOUSE CLICK INCREASE MODULE LEVEL
RIGHT MOUSE CLICK DECREASE MODULE LEVEL
LONG LEFT MOUSE KEY PRESS PUT MODULE IN HOME INVENTORY
LONG RIGHT MOUSE KEY PRESS DESTROY MODULE
CONTROL + LEFT MOUSE CLICK CHANGE MODULE LEVEL
CONTROL + RIGHT MOUSE CLICK ADD OR REMOVE MODULE TYPE TO IGNORE LIST
ALT + RIGHT MOUSE CLICK UPGRADE MODULE TYPE
ON INVENTORY MODULE
LEFT MOUSE CLICK INCREASE MODULE LEVEL
RIGHT MOUSE CLICK DECREASE MODULE LEVEL
LONG LEFT MOUSE KEY PRESS LOAD MODULE
LONG RIGHT MOUSE KEY PRESS DESTROY MODULE
LEFT MOUSE BUTTON DRAG SCROLL INVENTORY
ANY KEY CHANGE MODULE HOT KEY
CONTROL + LEFT MOUSE CLICK SPLIT MODULE
CONTROL + RIGHT MOUSE CLICK ADD OR REMOVE MODULE TYPE TO IGNORE LIST
ALT + LONG LEFT MOUSE KEY PRESS LOAD MODULE FOR GUARDIAN
ALT + RIGHT MOUSE CLICK UPGRADE MODULE TYPE
ALT + CONTROL + LEFT MOUSE CLICK SORT MODULES
ALT + CONTROL + RIGHT MOUSE CLICK MERGE MODULES

* = NOT YET IMPLEMENTED
